# Navigate to SAP Cloud Logging

- Home\
![Health Monitoring Overview](images/HM_Overview.png)


- Select Service Type “SAP BTP, Cloud Foundry environment”\
![Health Monitoring Services/Systems](images/HM_Service_Details.png)


- Select “SAPTechEd-HandsOn-CF”\
![Health Monitoring Metrics](images/HM_Metrics.png)


- Select Metric "Memory: Old Generation (Java)" and click on information\
![Health Monitoring Metric Information](images/HM_Metric_CloudLogging.png)


- Select “CLS Dashboard: OTel JVM Metrics”\
![Health Monitoring Metric Information](images/HM_CloudLogging_URL.png)


- Sign In to SAP Cloud Logging (account see chapter Cloud Identity Services)

- Result\
![Metric Cloud Logging](images/HM_CloudLogging.png)


- Hint: In case being asked to select your tenant: Select “Global”, confirm and open the link again


- Adjust Time Window\
![Metric Cloud Logging Filter](images/HM_CloudLogging_Filter.png)


- Result\
![Metric Cloud Logging Filtered](images/HM_CloudLogging_FilterApplied.png)
