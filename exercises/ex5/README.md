#  Synthetic User Monitoring

Synthetic User Monitoring is a 24/7 monitoring solution to detect performance and availability issues of applications from a client-side perspective.
The monitoring is based on:
-	Scenarios: sequence of actions simulating a real-user’s actions.
-	Runners: the infrastructure where the scenarios are executed. This represents the location of the simulated users.
Synthetic User Monitoring regularly executes the active scenarios on a set of associated runners.
Further information can be found on Expert Portal: [Synthetic User Monitoring](https://support.sap.com/en/alm/sap-cloud-alm/operations/expert-portal/synthetic-user-monitoring.html)
